# Work Overload & Stress - Global Survey Dashboard

Data analysis and visualization project exploring global patterns of work overload and stress using survey data.

The project combines Python-based statistical analysis (Google Colab) with an interactive Grafana dashboard to present regional trends, key drivers of overload, correlation structures, and anonymized raw responses.

## Project structure
- analysis/ - Python analysis and correlation calculations
- dashboard/ - Grafana dashboard screenshots
- data/ - Sample survey data
